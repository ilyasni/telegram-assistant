# Альтернативные способы получения Channel ID

## Проблема

Бот @userinfobot возвращает информацию о пользователе, а не о канале.

## Решения

### Способ 1: Бот @getidsbot

1. Откройте Telegram
2. Найдите бота @getidsbot
3. Отправьте ссылку на канал: `https://t.me/beer_for_all`
4. Бот должен вернуть Channel ID

### Способ 2: Бот @RawDataBot

1. Откройте Telegram
2. Найдите бота @RawDataBot
3. Перешлите любое сообщение из канала боту
4. Бот вернет raw данные, включая `peer_id.channel_id`

### Способ 3: Telegram Desktop (если есть доступ)

1. Откройте канал в Telegram Desktop
2. Перейдите в **View** → **Statistics** (если доступно)
3. Найдите **Channel ID**

### Способ 4: Через веб-версию Telegram

1. Откройте канал в web.telegram.org
2. Нажмите F12 (Developer Tools)
3. Перейдите на вкладку **Network**
4. Обновите страницу (F5)
5. Найдите запросы к API Telegram
6. В ответах найдите `channel_id` или `peer_id`

### Способ 5: Добавить канал через бота с telegram_id

Если у вас есть telegram_id канала, можно добавить его напрямую:

1. Откройте бота Telegram Assistant
2. Используйте команду `/add_channel`
3. Выберите "По Telegram ID"
4. Введите отрицательное число (например: `-1001234567890`)

### Способ 6: Использовать другой аккаунт/сессию

Если текущая сессия не имеет доступа к каналам, можно:
1. Использовать другую сессию Telegram с доступом к каналам
2. Получить ID через эту сессию
3. Обновить каналы в БД

## Рекомендация

Попробуйте **@getidsbot** или **@RawDataBot** - они специально предназначены для получения ID каналов и чатов.
