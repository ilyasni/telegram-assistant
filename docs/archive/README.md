# Архив временной документации

**Дата архивации:** 2025-02-01  
**Причина:** Временные статус-отчеты, фиксы и проверки, которые больше не актуальны

## Категории архивированных файлов

### Статус-отчеты
- `*_STATUS*.md` - временные статусы реализации
- `*_FINAL*.md` - финальные отчеты по завершенным задачам
- `*_REPORT*.md` - отчеты о проверках и диагностике

### Фиксы и исправления
- `*_FIX*.md` - отчеты об исправлениях багов
- `*_FIXES*.md` - сводки исправлений

### Тестирование
- `*_TEST*.md` - отчеты о тестировании
- `*_CHECK*.md` - отчеты о проверках

## Актуальная документация

Актуальная документация находится в корне `docs/`:
- `ARCHITECTURE_PRINCIPLES.md` - архитектурные принципы
- `SYSTEM_SPECIFICATION_v3.2.md` - спецификация системы
- `TESTING_GUIDE.md` - руководство по тестированию
- `DEPLOYMENT_CHECKLIST.md` - чеклист развертывания
- `SCRIPTS_INDEX.md` - индекс скриптов
- `CLEANUP_CHECKLIST.md` - чеклист очистки
